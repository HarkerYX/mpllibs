algebraic_tag
=============
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class T>
struct algebraic_tag
{
  // unspecified
};
--------------------------------------


Description
-----------

Lazy metafunction getting the algebraic tag of a value of an algebraic
data-type.

Header
------

--------------------------------------
#include <mpllibs/metamonad/algebraic_tag.hpp>
--------------------------------------

Example
-------

--------------------------------------
MPLLIBS_DATA(maybe, ((just, 1))((nothing, 0)));

static_assert(
  boost::is_same<maybe_tag, algebraic_tag<nothing>::type>::type::value,
  "Testing algebraic_tag"
);
