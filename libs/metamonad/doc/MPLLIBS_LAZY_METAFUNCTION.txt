MPLLIBS_LAZY_METAFUNCTION
=========================
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define MPLLIBS_LAZY_METAFUNCTION(name, body) \
  // unspecified
--------------------------------------


Description
-----------

This macro is similar to link:MPLLIBS_METAFUNCTION.html[+MPLLIBS_METAFUNCTION+]
but it wraps its +body+ with link:lazy.html[+lazy+].

Header
------

--------------------------------------
#include <mpllibs/metamonad/lazy_metafunction.hpp>
--------------------------------------

Expression semantics
--------------------

For any +body+ angly-bracket expression the following

--------------------------------------
MPLLIBS_LAZY_METAFUNCTION(name, (body));
--------------------------------------

is equivalent to

--------------------------------------
MPLLIBS_METAFUNCTION(name, (mpllibs::metamonad::lazy< body >));
--------------------------------------

Example
-------

--------------------------------------
using namespace boost::mpl;

template <class N>
MPLLIBS_LAZY_METAFUNCTION(double_value, (times<int_<2>, N>));
--------------------------------------
