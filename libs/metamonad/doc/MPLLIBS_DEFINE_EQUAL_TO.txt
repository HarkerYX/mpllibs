MPLLIBS_DEFINE_EQUAL_TO
=======================
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define MPLLIBS_DEFINE_EQUAL_TO(name) \
  // unspecified
--------------------------------------


Description
-----------

Macro for overloading +boost::mpl::equal_to+ to handle the algebraic data-type
+name+, defined by using link:MPLLIBS_DATA.html[+MPLLIBS_DATA+].
This macro has to be used in the top-level namespace.

Header
------

--------------------------------------
#include <mpllibs/metamonad/data.hpp>
--------------------------------------

Example
-------

--------------------------------------
namespace foo
{
  MPLLIBS_DATA(maybe, ((nothing, 0))((just, 1)));
}

MPLLIBS_DEFINE_EQUAL_TO(foo::maybe);
--------------------------------------
