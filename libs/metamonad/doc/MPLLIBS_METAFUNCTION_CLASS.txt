MPLLIBS_METAFUNCTION_CLASS
==========================
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define MPLLIBS_METAFUNCTION_CLASS(name, args) \
  // unspecified
--------------------------------------


Description
-----------

This macro is intended to be used for defining forwarding metafunction classes.
A macro call expands to metafunction forwarding using inheritance. +args+ is the
list of template arguments, each of them in parentheses. The +body+ is expected
to follow the macro call in double parentheses. +args+ can not be empty.

Header
------

--------------------------------------
#include <mpllibs/metamonad/metafunction.hpp>
--------------------------------------

Expression semantics
--------------------

For any +body+ angly-bracket expression, +n > 0+ and +arg1+ ... +argn+ template
arguments the following

--------------------------------------
MPLLIBS_METAFUNCTION_CLASS(name, (arg1)(arg2)...(argn)) ((body));
--------------------------------------

is equivalent to

--------------------------------------
struct name : mpllibs::metamonad::tmp_value<name>
{
  MPLLIBS_METAFUNCTION(apply, (arg1)(arg2)...(argn) ((body));
};
--------------------------------------

Example
-------

--------------------------------------
using namespace boost::mpl;

MPLLIBS_METAFUNCTION_CLASS(double_value, (N)) ((times<int_<2>, N>));

template <class N>
MPLLIBS_METAFUNCTION_CLASS(plusn, (M)) ((plus<N, M>));
--------------------------------------
