MPLLIBS_RETURN
==============
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define MPLLIBS_RETURN do_return

template <class Value>
struct do_return
  // unspecified
;
--------------------------------------


Description
-----------

Template expression that can be used inside a link:do_.html[do_]
block. +MPLLIBS_RETURN+ evaluates link:return_.html[return_]+<MonadTag, Value>+
using the +MonadTag+ of the do block containing +MPLLIBS_RETURN+.

Header
------

--------------------------------------
#include <mpllibs/metamonad/do.hpp>
--------------------------------------

Example
-------

--------------------------------------
using boost::mpl::int_;

struct r;

do_<exception_tag>::apply<
  MPLLIBS_RETURN<int_<13> >
>
--------------------------------------
