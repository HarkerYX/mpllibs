is_nothing
==========
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class Data>
struct is_nothing
{
  // unspecified
};
--------------------------------------


Description
-----------

Metafunction checking if its argument is a link:nothing.html[nothing] value. It
expects a nullary metafunction as argument.

Header
------

--------------------------------------
#include <mpllibs/metamonad/is_nothing.hpp>
--------------------------------------

Expression semantics
--------------------

For any +d+ nullary metafunction, the following are equivalent

--------------------------------------
mpllibs::metamonad::is_nothing<d>
boost::is_same<boost::mpl::tag<d::type>::type, nothing_tag>
--------------------------------------

Example
-------

--------------------------------------
typedef is_nothing<mpllibs::metaparse::nothing> always_true;
--------------------------------------
