MPLLIBS_LAZY_REC_METAFUNCTION
=============================
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define MPLLIBS_LAZY_REC_METAFUNCTION(name, body) \
  // unspecified
--------------------------------------


Description
-----------

This macro is similar to
link:MPLLIBS_REC_METAFUNCTION.html[+MPLLIBS_REC_METAFUNCTION+] but it wraps its
+body+ with link:lazy.html[+lazy+].

Header
------

--------------------------------------
#include <mpllibs/metamonad/lazy_metafunction.hpp>
--------------------------------------

Expression semantics
--------------------

For any +body+ angly-bracket expression the following

--------------------------------------
MPLLIBS_LAZY_REC_METAFUNCTION(name, (body));
--------------------------------------

is equivalent to

--------------------------------------
MPLLIBS_REC_METAFUNCTION(name, (mpllibs::metamonad::lazy< body >));
--------------------------------------

Example
-------

--------------------------------------
using namespace boost::mpl;

template <class N>
MPLLIBS_LAZY_REC_METAFUNCTION(
  fact,
  (
    eval_if<
      less<N, int_<1> >,
      int_<1>,
      times<fact<minus<N, int_<1> > >, N>
    >
  )
);
--------------------------------------
