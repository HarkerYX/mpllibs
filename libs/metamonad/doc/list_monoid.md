# list monoid

This is the C++ template metaprogramming equivalent of Haskell's List monoid.

## General information

* header: `<mpllibs/metamonad/list.hpp>`
* tag of monoid: `list_tag`
* `mempty`: empty list
* `mappend`: list concatenation
