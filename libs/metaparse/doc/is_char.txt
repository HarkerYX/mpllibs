is_char
=======
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class C>
struct is_char
{
  template <class S>
  struct apply
  {
    // unspecified
  };
};
--------------------------------------


Description
-----------

Checks if +C+ and +S+ are the same characters.

Header
------

--------------------------------------
#include <mpllibs/metaparse/util/is_char.hpp>
--------------------------------------

Expression semantics
--------------------

For any +c+ and +s+ wrapped characters

--------------------------------------
boost::mpl::apply<is_char<c>, s>
--------------------------------------

is equivalent to

--------------------------------------
boost::mpl::equal_to<c, s>
--------------------------------------

Example
-------

--------------------------------------
boost::mpl::apply<
  is_char<boost::mpl::char_<'a'> >,
  boost::mpl::char_<'a'>
>
--------------------------------------
