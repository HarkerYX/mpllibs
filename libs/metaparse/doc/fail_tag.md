# fail_tag

## Synopsis

```cpp
struct fail_tag
{
  // unspecified
};
```

## Description

This is the tag of the values returned by a parser when parsing was not
successful.

## Header

```cpp
#include <mpllibs/metaparse/fail_tag.hpp>
```

## Expression semantics

The following are equivalent

```cpp
fail_tag
fail_tag::type
```
