get_col
=======
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class source_position>
struct get_col
{
  // unspecified
};
--------------------------------------


Description
-----------

Returns the column of a source position.

Header
------

--------------------------------------
#include <mpllibs/metaparse/get_col.h>
--------------------------------------

Expression semantics
--------------------

For any +l+, +c+ compile-time wrapped integral values, the following are
equivalent

--------------------------------------
get_col<mpllibs::metaparse::source_position<l, c> >::type
c::type
--------------------------------------

Example
-------

--------------------------------------
typedef
  get_col<mpllibs::metaparse::start>::type
  one;
--------------------------------------
