nothing
=======
Abel Sinkovics

Synopsis
--------

--------------------------------------
struct nothing
{
  // unspecified
};
--------------------------------------


Description
-----------

Compile-time data-type representing parsing error. +nothing+ is
_equality comparable_. +nothing+ is equal to itself only.

It has a nested +type+ tag pointing to itself, +nothing+ can be used as a
nullary metafunction.

Header
------

--------------------------------------
#include <mpllibs/metaparse/nothing.h>
--------------------------------------

Expression semantics
--------------------

For any +a+ _equality comparable_ class, that is not +nothing+:

--------------------------------------
boost::mpl::equal_to<a, nothing>::type
boost::mpl::equal_to<nothing, a>::type
--------------------------------------

are both equivalent to +boost::mpl::false_+.

--------------------------------------
boost::mpl::equal_to<nothing, nothing>::type
--------------------------------------

is equivalent ot +boost::mpl::true_+.

+nothing::type+ is equivalent to +nothing+.
