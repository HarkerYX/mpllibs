compose
=======
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <
  class F1,
  // ...
  class Fn
>
struct compose
{
  // unspecified
};
--------------------------------------


Description
-----------

Higher order function taking metafunction classes as arguments and generating
their composition.

Header
------

--------------------------------------
#include <mpllibs/metaparse/util/compose.h>
--------------------------------------

Expression semantics
--------------------

For any +F1+ ... +Fn+ metafunction classes taking 1 argument and
+x+ class

--------------------------------------
boost::mpl::apply<compose<F1, .... Fn>, x>
--------------------------------------

is equivalent to

--------------------------------------
boost::mpl::apply<F1, ... boost::mpl::apply<Fn, x>::type ... >
--------------------------------------

Example
-------

--------------------------------------
struct make_pointer
{
  template <class T>
  struct apply : boost::mpl::identity<T*> {};
};

typedef
  boost::mpl::apply<compose<make_pointer, make_pointer>, int>
  int_ptr_ptr;

--------------------------------------
