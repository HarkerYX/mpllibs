make_pair
=========
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class a, class b>
struct make_pair
{
  // unspecified
};
--------------------------------------


Description
-----------

Metafunction taking two nullary metafunctions as arguments and building an
+mpllibs::metaparse::util::pair+ from them.

Header
------

--------------------------------------
#include <mpllibs/metaparse/util/make_pair.h>
--------------------------------------

Expression semantics
--------------------

For any +a+ and +b+ nullary metafunctions:

--------------------------------------
make_pair<a, b>::type
--------------------------------------

is equivalent to

--------------------------------------
mpllibs::metaparse::util::pair<a::type, b::type>
--------------------------------------

Example
-------

--------------------------------------
make_pair<
  boost::mpl::identity<int>,
  boost::mpl::identity<double>
>
--------------------------------------
