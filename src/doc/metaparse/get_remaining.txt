get_remaining
=============
Abel Sinkovics

--------

--------------------------------------
template <class D>

{
  // unspecified

Description
-----------

Returns the remaining string information of a parsing result.

Header
------

--------------------------------------
#include <mpllibs/metaparse/get_remaining.h>
--------------------------------------

Expression semantics
--------------------

For any +a+, +b+ and +c+ classes, the following are equivalent

--------------------------------------
get_remaining<mpllibs::metaparse::accept<a, b, c> >::type
b::type
