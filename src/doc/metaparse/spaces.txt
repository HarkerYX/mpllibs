spaces
======
Abel Sinkovics

Synopsis
--------

--------------------------------------
struct spaces
{
  template <class s>
  struct apply
  {
    // unspecified
  };
};
--------------------------------------


Description
-----------

Parser accepting any number of whitespace characters. It requires at least one
to be present. The result of parsing is undefined.

Header
------

--------------------------------------
#include <mpllibs/metaparse/spaces.h>
--------------------------------------

Expression semantics
--------------------

For any +s+ compile-time string

--------------------------------------
boost::mpl::apply<spaces, s>::type::second
--------------------------------------

is equivalent to

--------------------------------------
boost::mpl::apply<mpllibs::metaparse::any1<mpllibs::metaparse::space>, s>::type::second
--------------------------------------

when +s+ begins with a whitespace character. For any +s+ compile-time string,
that does not begin with a whitespace character

--------------------------------------
boost::mpl::apply<spaces, s>::type
--------------------------------------

is equivalent to

--------------------------------------
mpllibs::metaparse::nothing
--------------------------------------

Example
-------

--------------------------------------
typedef
  // ...
  some_string;

typedef
  boost::mpl::apply<spaces, some_string>
  some_string_without_leading_whitespaces;
--------------------------------------
