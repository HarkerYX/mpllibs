SET
===
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define SET set

template <class Name, class F>
struct set
  // unspecified
;
--------------------------------------


Description
-----------

Template expression that can be used inside a link:DO.html[DO] block.
+F+ is a nullary metafunction returning a monadic value. +SET+ binds the result
of +F+ to +Name+ in the rest of the do block. On how the binding works, see
link:let.html[let] expressions.

Header
------

--------------------------------------
#include <mpllibs/metamonad/do_.hpp>
--------------------------------------

Example
-------

--------------------------------------
using boost::mpl::int_;

struct r;

DO<exception_tag>::apply<
  SET<r, may_fail1<int_<13> >,
  may_fail2<r>
>
