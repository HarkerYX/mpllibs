RETURN
======
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define RETURN do_return

template <class Value>
struct do_return
  // unspecified
;
--------------------------------------


Description
-----------

Template expression that can be used inside a link:DO.html[DO] block.
+RETURN+ evaluates link:return_.html[return_]+<MonadTag, Value>+ using the
+MonadTag+ of the do block containing +RETURN+.

Header
------

--------------------------------------
#include <mpllibs/metamonad/do_.hpp>
--------------------------------------

Example
-------

--------------------------------------
using boost::mpl::int_;

struct r;

DO<exception_tag>::apply<
  RETURN<int_<13> >
>
