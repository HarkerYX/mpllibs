MPLLIBS_RETURN
==============
Abel Sinkovics

Synopsis
--------

--------------------------------------
#define MPLLIBS_RETURN do_return

template <class Value>
struct do_return
  // unspecified
;
--------------------------------------


Description
-----------

Template expression that can be used inside a link:MPLLIBS_DO.html[MPLLIBS_DO]
block. +MPLLIBS_RETURN+ evaluates link:return_.html[return_]+<MonadTag, Value>+
using the +MonadTag+ of the do block containing +MPLLIBS_RETURN+.

Header
------

--------------------------------------
#include <mpllibs/metamonad/do_.hpp>
--------------------------------------

Example
-------

--------------------------------------
using boost::mpl::int_;

struct r;

MPLLIBS_DO<exception_tag>::apply<
  MPLLIBS_RETURN<int_<13> >
>
--------------------------------------
