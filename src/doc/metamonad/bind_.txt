bind_
=====
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class MonadTag, class A, class B>
struct bind_
{
  // unspecified
};
--------------------------------------


Description
-----------

Takes a monad tag and two monadic values as arguments, returns a new monadic
value. It can be overloaded by +MonadTag+, its behaviour is dependent on it.
The default behaviour calls link:bind.html[bind] with +A+ as its monadic value
argument and a the constant +B+ function as its function argument.

Header
------

--------------------------------------
#include <mpllibs/metamonad/bind_.hpp>
--------------------------------------

Example
-------

--------------------------------------
struct one_error;
struct other_error;

typedef
  bind_<
    exception_tag,
    exception<one_error>,
    exception<other_error>
  >::type
  exception_one_error;
--------------------------------------
