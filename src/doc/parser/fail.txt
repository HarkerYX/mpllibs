fail
====
Abel Sinkovics

Synopsis
--------

--------------------------------------
struct fail
{
  template <class s>
  struct apply
  {
    // unspecified
  };
};
--------------------------------------


Description
-----------

Parser rejecting every input.

Header
------

--------------------------------------
#include <mpllibs/parser/fail.h>
--------------------------------------

Expression semantics
--------------------

For any +s+ compile-time string

--------------------------------------
boost::mpl::apply<fail, s>::type
--------------------------------------

is equivalent to +nothing+.
