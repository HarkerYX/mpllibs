digit
=====
Abel Sinkovics

Synopsis
--------

--------------------------------------
struct digit
{
  template <class s>
  struct apply
  {
    // unspecified
  };
};
--------------------------------------


Description
-----------

Parser accepting one character in the range +0-9+. The
result of the parser is the accepted character.

Header
------

--------------------------------------
#include <mpllibs/parser/digit.h>
--------------------------------------

Expression semantics
--------------------

For any +s+ compile-time string

--------------------------------------
boost::mpl::apply<digit, s>
--------------------------------------

is equivalent to

--------------------------------------
mpllibs::util::pair<boost::mpl::front<s>::type, boost::mpl::pop_front<s>::type>
--------------------------------------

when +boost::mpl::empty<s>::type+ is +false+ and +boost::mpl::front<s>::type+
is in the range +0-9+. It is +nothing+ otherwise.

Example
-------

--------------------------------------
boost::mpl::apply<mpllibs::parser::digit, boost::mpl::string<'13'> >
--------------------------------------
