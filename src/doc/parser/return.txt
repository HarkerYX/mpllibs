return
======
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class c>
struct return_
{
  template <class s>
  struct apply
  {
    // unspecified
  };
};
--------------------------------------


Description
-----------

Parser accepting every input. The result of parsing is +c+, the remaining string
is the input string.

Header
------

--------------------------------------
#include <mpllibs/parser/return.h>
--------------------------------------

Expression semantics
--------------------

For any +c+ class and +s+ compile-time string

--------------------------------------
boost::mpl::apply<return_<c>, s>
--------------------------------------

is equivalent to

--------------------------------------
mpllibs::util::pair<c, s>
--------------------------------------

Example
-------

--------------------------------------
typedef return_<int> always_int;
--------------------------------------
