digit_val
=========
Abel Sinkovics

Synopsis
--------

--------------------------------------
struct digit_val
{
  template <class s>
  struct apply
  {
    // unspecified
  };
};
--------------------------------------


Description
-----------

Parser accepting one character in the range +0-9+. The result of the parser is
the value represented by the accepted character.

Header
------

--------------------------------------
#include <mpllibs/parser/digit_val.h>
--------------------------------------

Expression semantics
--------------------

For any +s+ compile-time string

--------------------------------------
boost::mpl::apply<digit, s>
--------------------------------------

is equivalent to

--------------------------------------
mpllibs::util::pair<
  mpllibs::util::digit_to_int::apply<boost::mpl::front<s>::type>::type,
  boost::mpl::pop_front<s>::type
>
--------------------------------------

when +boost::mpl::empty<s>::type+ is +false+ and +boost::mpl::front<s>::type+
is in the range +0-9+. It is +nothing+ otherwise.

Example
-------

--------------------------------------
boost::mpl::apply<mpllibs::parser::digit_val, boost::mpl::string<'13'> >
--------------------------------------
