equal_sequence
==============
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class a, class b>
struct equal_sequence
{
  // unspecified
};
--------------------------------------


Description
-----------

Compares two sequences element-by-element. They are equal when they have the
same length and every element of one of the sequences is equal to the element
in the other sequence at the same location. Elements are compared using
+boost::mpl::equal_to+.

Header
------

--------------------------------------
#include <mpllibs/metatest/equal_sequence.h>
--------------------------------------

Expression semantics
--------------------

For every +a+ and +b+ sequences:

--------------------------------------
equal_sequence<a, b>
--------------------------------------

is equivalent to

--------------------------------------
boost::mpl::and_<
  boost::mpl::equal_to<
    boost::mpl::front<a>,
    boost::mpl::front<b>
  >,
  
  boost::mpl::equal_to<
    boost::mpl::front<boost::mpl::pop_front<a>::type>,
    boost::mpl::front<boost::mpl::pop_front<b>::type>
  >,
  
  // ...
  
  boost::mpl::equal_to<

    boost::mpl::front<
      boost::mpl::pop_front<
      // ...
      boost::mpl::pop_front<
        a
      >::type
      // ...
      >::type
    >,
    
    boost::mpl::front<
      boost::mpl::pop_front<
      // ...
      boost::mpl::pop_front<
        b
      >::type
      // ...
      >::type
    >
    
  >,
  
  // ... and none of the sequences was shorter than the other one
>
--------------------------------------

Example
-------

--------------------------------------
equal_sequence<
  boost::mpl::vector<
    boost::mpl::int_<11>,
    boost::mpl::int_<13>
  >,
  boost::mpl::list<
    boost::mpl::int_<11>,
    boost::mpl::int_<13>
  >
>::type
--------------------------------------
