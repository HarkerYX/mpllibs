swap
====
Abel Sinkovics <abel@elte.hu>

Synopsis
--------

--------------------------------------
template <class p>
struct swap
{
  // unspecified
};
--------------------------------------


Description
-----------

Swaps the elements of a pair. Regardless of the type of the input pair,
the value of the metafunction is always an +mpllibs::util::pair+.

Header
------

--------------------------------------
#include <mpllibs/util/swap.h>
--------------------------------------

Expression semantics
--------------------

For any +p+ class, that has +p::first+ and +p::second+ member classes:

--------------------------------------
swap<p>::type
--------------------------------------

is equivalent to

--------------------------------------
mpllibs::util::pair<p::second, p::first>
--------------------------------------

Example
-------

--------------------------------------
swap<mpllibs::util::pair<int, double> >
--------------------------------------
