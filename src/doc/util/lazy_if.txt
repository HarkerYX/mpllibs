lazy_if
=======
Abel Sinkovics <abel@elte.hu>

Synopsis
--------

--------------------------------------
template <class c, class t, class f>
struct lazy_if
{
  // unspecified
};
--------------------------------------


Description
-----------

+boost::mpl::if_+ with lazily evaluated condition.

Header
------

--------------------------------------
#include <mpllibs/util/lazy_if.h>
--------------------------------------

Expression semantics
--------------------

For any +c+ nullary metafunction evaluating to a wrapped boolean value and
any +t+ and +f+ classes:

--------------------------------------
lazy_if<c, t, f>
--------------------------------------

is equivalent to

--------------------------------------
boost::mpl::if_<c::type, t, f>
--------------------------------------

Example
-------

--------------------------------------
boost::mpl::apply<
  lazy_if<
    boost::mpl::equal_to<boost::mpl::int_<1>, boost::mpl::int_<2> >,
    boost::mpl::int_<11>,
    boost::mpl::int_<13>
  >
>
--------------------------------------
