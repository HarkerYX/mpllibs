pair
====
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class a, class b>
struct pair
{
  // unspecified
};
--------------------------------------


Description
-----------

Extension of +boost::mpl::pair+: these pairs can be compared using
+boost::mpl::equal_to+. Two pairs are equal when their elements are equal.

Header
------

--------------------------------------
#include <mpllibs/util/pair.h>
--------------------------------------

Expression semantics
--------------------

For any +a1+, +b1+, +a2+ and +b2+ _equality comparable_ classes:

--------------------------------------
boost::mpl::equal_to<pair<a1, b1>, pair<a2, b2> >
--------------------------------------

is equivalent to

--------------------------------------
boost::mpl::and_<
  boost::mpl::equal_to<a1, a2>::type,
  boost::mpl::equal_to<b1, b2>::type
>
--------------------------------------

Example
-------

--------------------------------------
pair<int, double>
--------------------------------------
